Цикломатска комплексност
- V(G) = E - N + 2 формулата за цикломатската комплексност , каде Е - е бројот на стрелки ( edges ), а N - е бројот на кругови ( nodes ).
од нацртаниот CFG ( Control Flow Graph ) можеме да пресметаме. V(G) = 16 - 16 + 2 = 2 .
